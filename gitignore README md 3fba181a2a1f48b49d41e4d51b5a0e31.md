# gitignore

- ❤ **깃허브에 올리지 말아야 할 파일 목록을 작성 후 거를 수 있다.**
- ❤ git으로 관리하는 최상위 폴더 하위에 **.gitignore** 이란 파일로 넣는다. **확장자 없음**
- 와일드 카드 문자를 활용 → 구글에 검색
- **💥꿀팁💥 폴더 하나 생성하고 올라가면 안되는 애들을 그 폴더에 넣고 /폴더 하면 다 안올라간다.**
