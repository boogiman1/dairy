# textarea, cols, rows

텍스트 입력 영역을 만들어준다.

❤cols : 속성안에 들어간 숫자만큼의 가로 영역을 만들어준다. 텍스트 입력 영역 중 보이는 영역의 너비를 명시한다.

❤rows : 속성안에 들어간 숫자만큼의 줄 영역을 만들어준다. 줄이 넘어가면 자동으로 스크롤이 생김
